Mini Python Browser - 2A Release Setup
======================================

Welcome to the Mini Python Browser 2A Setup!

Files in this folder:
---------------------
- SetUp2A.py         -> The main setup script for the browser
- RunSetup2A.bat     -> Launcher to run the setup on Windows
- README.md          -> This file

How to Install / Run:
---------------------
1. Make sure Python is installed on your computer. 
   - If Python is not installed, download it from: https://www.python.org/downloads/

2. Run the setup:
   - Double-click 'RunSetup2A.bat' to launch the setup.

3. Follow the on-screen instructions to install the Mini Python Browser.
