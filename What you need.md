NEED

This project requires the following Python modules and packages:

1. Standard libraries (included with Python):
   sys, os, json, shutil, datetime, traceback

2. External packages (install via pip):
   - cryptography (Fernet)
   - PyQt6 (GUI & browser support)
   
3. PyQt6 modules used:
   - QtWidgets: QApplication, QMainWindow, QVBoxLayout, QWidget, QLineEdit, QPushButton, QHBoxLayout, QTabWidget, QMessageBox, QDialog, QListWidget, QLabel, QInputDialog, QFileDialog
   - QtGui: QIcon, QPixmap, QColor, QAction
   - QtCore: QUrl, QObject, pyqtSlot, QTimer
   - QtWebEngineWidgets: QWebEngineView
   - QtWebEngineCore: QWebEngineProfile
   - QtWebChannel: QWebChannel

Make sure to run the following commands in your terminal before running the project:

pip install cryptography

pip install PyQt6 PyQt6-WebEngine
